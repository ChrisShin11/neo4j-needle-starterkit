# Neo4j Needle StarterKit

Responsive Starter Kit for building an app using [Neo4j Needle](https://www.neo4j.design/) for accelerating TTV.

Features:
1. Responsive App bar
2. Light/Dark mode toggle with theme wrapper
3. Side Navigation
